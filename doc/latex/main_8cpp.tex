\doxysubsection{Detailed Description}
The complete program of the dynamic queue. Includes a menu function so that the user may input any desired value. 

The options are as followed:\+
\begin{DoxyItemize}
\item Option 1 allows the user to input any value of type int.
\item Option 2 allows the user to delete the \doxylink{class_node}{Node} at the front of the queue.
\item Option 3 allows the user to view the value at the front of the queue.
\item Option 5 allows the user to view all the Nodes of the queue, as well as the first and last elements.
\item Option 5 allows the user to exit the program. 
\end{DoxyItemize}
