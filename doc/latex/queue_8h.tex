\doxysubsection{Detailed Description}
Defines the structure of the \doxylink{class_queue}{Queue}. Its structure consists of two pointers called "{}head"{} and "{}tail"{}, respectively. 

A queue is a linear data structure that follows the FIFO (First-\/in, First-\/out) method, thus making the core of its functions the first (front) and last (back) elements, or in this case Nodes, added to the queue. This file defines the constructors of the functions of the queue.
