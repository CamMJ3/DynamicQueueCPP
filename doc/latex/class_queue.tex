Deletes the \doxylink{class_node}{Node} at the front of the queue. 

If the queue is empty, then an error message is thrown. Otherwise, a new pointer "{}delete\+Node"{} that references to the front of the queue (pointer "{}head"{}) is initialized, while another variable stores the value stored in "{}head"{} to return it.

The front of the queue becomes the \doxylink{class_node}{Node} after the previous head. Finally, "{}delete\+Node"{} is deleted.
