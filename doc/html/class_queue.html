◆ dequeue()

template<typename T>
T Queue< T >::dequeue ( )

Deletes the Node at the front of the queue.

If the queue is empty, then an error message is thrown. Otherwise, a new pointer "deleteNode" that references to the front of the queue (pointer "head") is initialized, while another variable stores the value stored in "head" to return it.

The front of the queue becomes the Node after the previous head. Finally, "deleteNode" is deleted.

Returns
The value stored in the deleted Node.

< If "head" is nullptr, so will "tail" be, meaning the queue is now empty.

Prints all Nodes in the queue, highlighting the value at the front and back of the queue.

If the queue is empty, then an error message is thrown. Otherwise, a new pointer "tmp" that references to the front of the queue (pointer "head") is initialized. template<typename T>
void Queue< T >::enqueue ( Node< T > * newNode)

Adds a new Node, with a value, to the back of the queue.

If the queue is empty, the new Node becomes both the head and tail of the stack. Otherwise, the pointer of the new Node is set to the memory address of "head", then the new Node becomes the back of the queue.
